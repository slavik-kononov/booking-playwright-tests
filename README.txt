Automate the core car rental flow:

a) Run Tests via GitHub Actions:

2) Open the repository on GitHub.
3) Navigate to Actions tab.
4) Select Playwright Tests workflow.
5) Click Run workflow.
7) Click Run workflow.


b) Download Playwright Report:
1) Open the completed workflow run.
2) Scroll to the Artifacts section.
3) Download the artifact named:
playwright-report
4) Extract the archive.
5) Open the report playwright-report-{index}/index.html in any browser.
